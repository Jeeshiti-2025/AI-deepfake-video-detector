# AI Deepfake Video Detection System

## Problem Statement
The misuse of Artificial Intelligence has enabled the creation of deepfake videos that violate privacy, spread misinformation, and generate misleading or non-consensual content. Detecting such videos is difficult for common users.

## Solution
An AI-based video detection system that analyzes uploaded videos and determines whether the video is AI-generated or real.  
The MVP is implemented using Google Gemini AI.

## MVP Link
https://g.co/gemini/share/811bfaf71757

## Features
- AI-generated video detection
- Simple video upload workflow
- Clear result output (Real / AI-generated)
- User-friendly interface
- Privacy-focused analysis

## Workflow
1. User uploads a video
2. Video is analyzed using Gemini AI
3. AI checks for deepfake patterns
4. Result is displayed to the user
